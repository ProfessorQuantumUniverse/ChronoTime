## ğŸ”® About The App

**ChronoTime** is not just a clock; it is a kinetic sculpture of time for your Android device. 

Built with a stunning **Liquid Glass** design language, the app utilizes **AGSL Shaders** to create living, breathing backgrounds that react to your touch. Every interaction is paired with rich, immersive **haptic feedback**, making time feel tangible.

Forget standard ticking. Swipe vertically to travel through different dimensions of time measurement, from historical artifacts like the *Berlin Clock* to developer-centric *Unix Epochs*.
